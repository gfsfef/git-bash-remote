Git bash

Comado Git Exemplo de uso Operação
config git config --global user.name | user.email Configura o usuario e e-mail
help git help <comando> Obtém ajuda sobre um comando especifico
init git init Inicializar um repositorio em uma pasta
status git status Verificar a situação (status) do repositorio
add <file> git add. | git add <arquivo> Adiciona um mais arquivos ao string area
commit git commit -m "mensagem" Cria a versão (commita) o repositorio
reset git reset <arquivo> Volta do estado preparado para modificado
restore git <restore> Desfaz uma alteração em um arquivo modificado
clean git clean -fd Desfaz a criação (apaga) um arquivo criado
branch git branch -M "main" Mudar o nome da branch principal
code code . | code <arquivo> Abre o VSCode ou na pasta ou um arquivo
